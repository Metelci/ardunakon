# Arduino GIGA R1 WiFi Setup Guide

Setup for the high-performance Arduino GIGA R1 WiFi with STM32H7.

## Overview

| Feature | Specification |
|---------|--------------|
| **MCU** | STM32H747XI (Dual Cortex-M7/M4) |
| **Wireless** | Murata 1DX (WiFi + BLE) |
| **Voltage** | 3.3V logic (5V tolerant on most pins) |
| **PWM Pins** | Most digital pins |
